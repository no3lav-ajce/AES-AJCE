# AJCE Professional Portal

A modern, institutional web portal for **Amal Jyothi College of Engineering (AJCE)** — designed with a premium codebase reflecting academic excellence and technical innovation.

---

## 🚀 Key Features

### 1. Interactive Virtual Tour
A dedicated application that allows prospective students to explore the campus through an immersive experience, featuring pulsing hotspots with real-time building information.

### 2. Live Program Explorer
A dynamic course discovery tool with:

- **Real-time search** – Find programs using keywords  
- **Category filtering** – Switch between B.Tech, M.Tech, and Research programs  
- **Detailed data** – Integrated seat counts and syllabus access

### 3. Premium Design System

- **Refined Academic Theme** – Slate Navy and Crimson color palette for a premium institutional look  
- **Responsive layout** – Fluid grid system for mobile and desktop  
- **Glassmorphism UI** – Modern overlay effects and subtle micro-animations

---

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla CSS3 (custom design tokens)
- **Styling**: Utility-first grid system and component-based CSS
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Typography**:  
  - Playfair Display – headings  
  - Inter – body text
